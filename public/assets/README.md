# Assets Directory

This directory contains the image assets required for the landing page.

## Required Image Files

The following image files need to be placed in this directory for the landing page to display correctly:

### Background Images
- `433.svg` - Main background rectangle (1728x1117)
- `6214.svg` - Ellipse background element (566x451) - blurred
- `13538.svg` - Large ellipse background (1022x1000) - blurred

### UI Elements
- `7227.svg` - Button background rectangle (348x65)
- `1246.svg` - Small rectangle element (80x70)
- `1186.svg` - Frame/icon element (44x41)

### Content Images
- `7242.png` - Rectangle content image (203x144)
- `7261.png` - Rotated rectangle content (552.62x514)
- `7258.png` - Large rectangle content (985x1091.27)
- `629.png` - Inner rectangle content (1091x721)

## Notes
- All images should be optimized for web use
- SVG files should maintain their vector properties
- PNG files should be optimized for file size while maintaining quality
- The landing page will gracefully handle missing images by showing broken image placeholders

## Image Sources
These images appear to be design assets that should be provided by the design team or extracted from the original design files.
